Shape Community App is a public frontend for browsing geometric toy shapes, exploring individual shape pages, and later adding compatibility testing, 3D previews, and live activity signals.

## Stack

- React for the UI
- TypeScript for type safety
- React Router for client-side routing
- Vite as the local dev server and production bundler
- pnpm for package management
- Node 24 for local development and builds

Vite is developer tooling, not runtime app infrastructure. The deployed artifact is a static frontend bundle generated by `vite build`.

## Planned Routes

- `/`
- `/shapes`
- `/shapes/:slug`

## Getting Started

```bash
pnpm install
pnpm dev
```

## Production Build

```bash
pnpm build
```

The production output will be a static bundle that can be deployed behind Nginx or any static hosting setup.
